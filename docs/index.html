scrilla

Overview

Welcome to the documentation page for scrilla. scrilla is a financial analysis library written for Python. It retrieves data from external services and produces statistical summaries. It can be used for a variety of functions, such as calculating asset correlations, optimizing portfolios, asset probability calculations, calibrating price models to historical data, evaluating the soundness of models and much more. Install it directly from the distribution on PyPi, pip install scrilla Checkout out the README or the links below for more information on setting up and configuring your scrilla installation. The content on this page is generated through a CI/CD pipeline and hosted statically through Github.
